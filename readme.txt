Display related posts in a beautiful, responsive carousel layout using Elementor page builder.

== Description ==

Related Posts Elementor Widget enhances your WordPress site with a powerful and flexible way to showcase related content. Built specifically for Elementor, this widget offers a modern carousel layout with advanced customization options.

= Key Features =

* Smart Carousel Layout
* Container-Based Responsive Design
* Advanced Post Query Options
* Rich Content Display
* Fully Customizable Navigation
* Mobile-First Approach

= Smart Carousel Layout =

* Responsive Swiper-based carousel
* Container-based breakpoints for better layout control
* Customizable slides per view based on container width
* Smooth transitions and navigation

= Advanced Query Options =

* Display related posts based on categories
* Support for custom post types
* Configurable number of posts to show
* Custom post ordering

= Rich Content Display =

* Featured image backgrounds
* Customizable content overlay
* Adjustable excerpt length
* Call-to-action buttons

= Fully Customizable Design =

* Configurable overlay opacity and gradient
* Custom navigation colors and styles
* Responsive typography settings
* Flexible layout options

= PRO Features =

* Advanced post filtering
* Custom taxonomies support
* Additional carousel effects
* Priority support

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/related-posts-elementor` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Edit a page with Elementor
4. Search for "Related Posts" in the Elementor editor
5. Drag and drop the widget into your layout
6. Configure the widget settings through the provided options

== Frequently Asked Questions ==

= Does this plugin require Elementor? =

Yes, this plugin requires Elementor Page Builder to be installed and activated.

= Is it responsive? =

Yes! The widget uses a container-based responsive design that adapts to different screen sizes and container widths.

= Can I customize the colors and typography? =

Absolutely! The widget provides extensive customization options for colors, typography, spacing, and more through the Elementor interface.

= Does it work with custom post types? =

Yes, the widget supports both default WordPress posts and custom post types.
